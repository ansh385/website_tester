# 🧪 TestNova

TestNova is a simple **Website Testing Tool** that helps testers check different testing tasks for a website.

It is designed to simulate the work of a **website tester** by organizing multiple types of software testing in one place.

## 🚀 Features

- Functional Testing
- Unit Testing
- Integration Testing
- System Testing
- UI Testing
- Performance Testing
- Security Testing
- Compatibility Testing

## 📂 Project File

```
TestNova.html
```

## 🛠️ How to Use

1. Download or clone this repository.
2. Open `TestNova.html` in your web browser.
3. Use it to explore different types of website testing tasks.

## 💻 Technologies Used

- HTML
- CSS
- JavaScript

## 🎯 Purpose

The purpose of this project is to **practice website testing concepts and simulate the work of a website tester**.  
It helps understand how different testing types are used to check the quality of a website.
